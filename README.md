# Setup
```bash
$ git clone https://github.com/Things-Network/Things-Proxy.git
$ npm install
$ npm start
```
The port is set to ``8080`` on default, and ``443`` when you have included a SSL Certificate, but you can customize it within your ``.env`` file.
